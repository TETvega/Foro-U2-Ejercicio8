Cómo utilizar la Listadora

Crea una cuenta

Para crear una cuenta, ingresa al apartado de inicio de sesión, llena el formulario con los datos de Nombre, correo electronico, contraseña y presiona el botón de crear.

Recuerda que solo puedes tener una cuenta por correo electrónico.

Iniciar sesión

Asegúrate de contar con una cuenta. Ingresa tu correo electrónico y la contraseña que utilizaste.

Creación y Eliminación de Productos

Cómo crear un Producto

Cuentas con un formulario a tu lado derecho en el que ingresarás los datos del producto, así como también su categoría. Para productos que ya existen o cuando cuentas con categorías existentes, se listarán y podrás seleccionarlas automáticamente. Asegúrate de que todos los campos estén llenos.

Cómo Eliminar o Editar un Producto

Cuando tienes productos existentes, puedes darle en su botón de editar. Este cargará los datos al formulario desde el cual podrás editar los datos o eliminar el producto. Si el producto era el único con una categoría, esta categoría se eliminará automáticamente.

USOS COMO SUDO ADMIN

Ingresar como SUDO

Para ingresar como sudo, ingresa como nombre de usuario "SUDO" y como contraseña "123" .

Respecto al Proyexto

Aplicación de Lista de Compras

Funcionalidades Principales

Agregar Productos: Los usuarios pueden agregar productos a su lista de compras. Esto implica ingresar el nombre del producto y la cantidad necesaria.

Eliminar Productos: Se puede eliminar productos de la lista de compras si ya no son necesarios.

Almacenamiento con LocalStorage: La lista de compras del usuario se almacena utilizando LocalStorage, lo que garantiza que la información persista incluso después de cerrar el navegador.

Funcionalidades Adicionales

Marcar Productos como Comprados: Los usuarios pueden marcar los productos de su lista como "comprados" una vez que los hayan adquirido, lo que les permite llevar un seguimiento de lo que ya han adquirido y lo que aún necesitan comprar.

Categorización de Productos: Existe la opción de categorizar los productos por tipo, como alimentos, productos de limpieza, etc. Esto facilita la organización de la lista y permite a los usuarios encontrar rápidamente los productos que están buscando.

Persistencia de Datos

Una de las características clave de la aplicación es que la información de la lista de compras se guarda localmente en el dispositivo del usuario utilizando LocalStorage. Esto significa que los usuarios pueden acceder a su lista de compras en cualquier momento y desde cualquier lugar, sin preocuparse por perder sus datos si cierran el navegador o apagan el dispositivo.

Interfaz Amigable

La aplicación debe contar con una interfaz intuitiva y fácil de usar, que permita a los usuarios agregar, eliminar y gestionar productos de manera sencilla y rápida.
